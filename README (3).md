\# Data Analytics Project 2 - Exploratory Data Analysis (EDA)



\## Internship

DecodeLabs Data Analytics Internship



\## Project Overview



This project focuses on Exploratory Data Analysis (EDA) using a cleaned dataset prepared during Project 1.



The purpose of this project is to analyze data, discover hidden patterns, identify trends, understand distributions, detect outliers, and generate meaningful insights using Python data analysis techniques.



\## Objective



The main objectives of this project are:



\- Explore the cleaned dataset

\- Understand data structure and distributions

\- Calculate descriptive statistics

\- Identify trends and patterns

\- Detect outliers

\- Analyze relationships between variables

\- Generate useful insights from data



\## Dataset



The dataset used in this project is the cleaned dataset created during Project 1 (Data Cleaning and Preparation).



The cleaned dataset was used for Exploratory Data Analysis to ensure accurate analysis and visualization.



\## Tools and Technologies Used



\- Python

\- Google Colab

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn



\## Analysis Performed



\### Data Understanding



Performed initial exploration:



\- Checked dataset shape

\- Viewed sample records

\- Checked column names

\- Analyzed data types

\- Checked missing values



\### Descriptive Statistics



Calculated:



\- Count

\- Mean

\- Median

\- Minimum values

\- Maximum values

\- Standard deviation



\### Data Visualization



Created visualizations including:



\- Histograms

\- Bar charts

\- Box plots

\- Correlation heatmap



\### Outlier Detection



Used boxplots and statistical methods to identify unusual values and possible outliers.



\### Correlation Analysis



Created a correlation matrix and heatmap to analyze relationships between numerical variables.



\## Key Insights



\- Data distributions were analyzed using visualizations.

\- Important trends and patterns were identified.
