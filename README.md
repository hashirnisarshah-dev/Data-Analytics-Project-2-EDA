# Data Analytics Project 2 - Exploratory Data Analysis (EDA)

## Internship
DecodeLabs Data Analytics Internship

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) to understand the hidden patterns, trends, distributions, and relationships present in a dataset.

The goal of this project is to transform raw data into meaningful insights using statistical analysis and visualization techniques.

## Objective

The main objectives of this project are:

- Explore the dataset structure
- Perform basic data analysis
- Calculate descriptive statistics
- Identify trends and patterns
- Detect outliers
- Analyze relationships between variables
- Generate meaningful insights

## Dataset

The dataset used in this project is the raw dataset from Project 1.

The dataset was analyzed using Python libraries and visualization techniques.

## Tools and Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed

### Data Understanding
- Checked dataset shape
- Viewed sample records
- Checked column information
- Examined data types
- Checked missing values and duplicates

### Statistical Analysis

Calculated:

- Count
- Mean
- Median
- Minimum values
- Maximum values
- Standard deviation

### Data Visualization

Created visualizations including:

- Histograms
- Bar charts
- Box plots
- Correlation heatmap

### Outlier Detection

Identified unusual data points using boxplots and statistical analysis.

### Correlation Analysis

Analyzed relationships between numerical variables using correlation matrix and heatmap.

## Key Insights

- Dataset distributions were analyzed.
- Important trends were identified.
- Outliers were detected and studied.
- Relationships between variables were explored.
